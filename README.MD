# Narrowly-POSIX complient UNIX tools
This project aims to be as POSIX complient as possible.<br>
There are no extra options than described by POSIX.<br>

## Currently finished programs
- echo
- cat
- tee
- true
- false
- sleep
- unlink
- head
- mv
- rmdir
- basename
- dirname
- id
- uname
- wc
- rm
- tr
